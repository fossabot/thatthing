Install an app

{{.}}
Once installed, you cannot change any of this. Unless you edit the db file. Or you reinstall it. You can change "public" in the app.json file, to privatize/publicize the app. If an app is private, the app can only be accessed when you're logged in. When an app is uninstalled, their data may not be removed automatically, although you can remove it manually. Restart ThatThing after installing an app.
